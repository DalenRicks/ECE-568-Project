## Responsibilities:

- compute skew

## Tasks:

2) Compute

$ x_i = t_i - t_1$
​
$w_i = \frac{T_i - T_1}{Hz}$
​
3) Compute offset

$y_i = w_i - x_i$

4) Plot

`offset vs time`

5) Estimate skew slope

$y = αx + β$

6) Save slopes from different ESP32 devices with the MAC address.

```
Device A MAC → skew +3 ppm
Device B MAC → skew -2 ppm
Device C MAC → skew +1 ppm
```
